# Shares tkchart

## 简述

这是一个绘制技术图形的python库，配合matplotlib库（稳定）和plotly库（存在性能问题）实现可视化

### 特色

- 能够自定义各种回测策略，基于python的条件表达式列表
- 在能够（实时）获取数据的条件下，进行策略选股
- 在有（实时）数据的条件下，实现基于数据时间精度的可视化

## 问题

- 完全剥离股票类和图表的文件关联, 单独设置一个作图系统和股票回测系统

## 回测指标

- 最大资金使用率
- 测试天数
- 测试周期数
- 信号个数
- 初始资金
- 最大资金利用率
- 杠杆倍数
- 收益
- 收益率
- 年化单利收益率
- 夏普比率
- 权益最大回撤
- 权益最大回撤比率
- 胜率
- 盈亏比率
- 最大每手盈利
- 最大每手亏损
- 手续费
- 手续费/收益
- vwap算法、twap算法
